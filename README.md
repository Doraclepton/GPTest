# GPTest - Описание
GPTest - программа работающая с кодом, развитие программы будет продолжатся!
# GPTest_0.0.1 
| То-что | Есть          |
| ------- | ------------------ |
| В программе: | Сканирование написанного кода на Python, вывод ссылок на официльные сайты Python по библиотекам/модулям в коде и разбор кода программой. Несколько локальных сайтов где объясняется пользование программой. |
| Обработка кода: | При обработке кода программа в первую очередь ищет библиотеки/модули и при их нахождении программа даёт нам ссылку на официальные ресурсы по ним. Коментарии начинаются и заканчиваются символом # . В полном отчёте программа не сможет дать разумный ответ по библиотекам/модулям, программа способна обрабатывать Python код. |
| Недочёты | При закрытии окна где выведен результат обработки кода, окно уже нельзя будет открыть без перезахода в программу. |
